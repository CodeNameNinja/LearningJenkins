Tutorial: Create Jenkins Compose File

Create Jenkins Compose File

Instructions for Docker Compose file management

  1. mkdir jenkins

pwd
/home/jenkins/

  1. cd jenkins

  2. vi docker-compose.yml

  3. copy and paste the following.

version: '3'
services:
  jenkins:
    container_name: jenkins
    image: jenkins/jenkins
    ports:
      - '8080:8080'
    volumes:
      - "$PWD/jenkins_home:/var/jenkins_home"
    networks:
      - net
networks:
  net:
  1. esc type: wq

verify that it saved

  1. cat docker-compose.yml

  2. mkdir jenkins_home

  3. docker-compose up -d

that will spin the jenkins server in detached mode
